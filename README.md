# Electric-Vehicle-Adoption-Analysis-Washington-State
This project analyzes EV adoption in Washington State by comparing BEVs and PHEVs over time. Using official registration data, the dataset was cleaned, filtered, and aggregated in Python. A stacked bar chart was improved to a stacked area chart, revealing rapid BEV growth after 2018 and its dominance in EV adoption.

Dataset

Washington State Electric Vehicle Population Data
https://data.wa.gov/Transportation/Electric-Vehicle-Population-Data/f6w7-q2d2

Tools Used

Python, Pandas, Matplotlib
